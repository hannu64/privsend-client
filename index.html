    Files are destroyed a little differently from text. Text is destroyed the moment it is read. If we destroyed a file the instant the download started, an interrupted download would leave the recipient with nothing and no way to try again. So a file is destroyed as soon as the recipient's browser confirms it decrypted successfully — usually within seconds — and in any case no later than 60 minutes after they open the link.

    The two passphrases do not match.

    Type it carefully. If you mistype the passphrase, nobody can ever decrypt this secret — not the recipient, not us. There is no recovery.
    Send the passphrase by a different channel than the link (e.g. link by email, passphrase by phone). If both travel together, it adds nothing.
